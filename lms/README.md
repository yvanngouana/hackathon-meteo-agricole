# Résultats Quiz LMS

## Instructions

Déposer les captures d'écran ou PDFs des résultats du quiz LMS pour chaque membre de l'équipe.

Format suggéré :
- `quiz_membre1.pdf` ou `quiz_membre1_screenshot.png`
- `quiz_membre2.pdf` ou `quiz_membre2_screenshot.png`
- `quiz_membre3.pdf` ou `quiz_membre3_screenshot.png`

## Informations requises

Chaque preuve doit montrer :
- Nom du participant
- Score obtenu
- Date de passage
- Validation du quiz

**Note** : Les résultats du quiz LMS sont obligatoires pour la validation du hackathon.
